# Nepra

**Nepra** is a static, multipage frontend website built with HTML, CSS, and JavaScript.  
It features responsive design, structured navigation, and visually distinct content sections for demonstrating foundational frontend development skills.

## What It Does
This project delivers a clean, professional website layout with multiple linked pages and UI elements that are styled and responsive without using frameworks. 

##  Project Structure

- `index.html` — Main landing/home page  
- `about.html`, `activities.html`, `tenable.html` — Supporting content pages  
- `css/` — Stylesheets for layout, typography, responsiveness  
- `js/` — JavaScript logic (e.g., menu toggles, interactions)  
- `images/` — All images and assets used in the site  
- `partners/` & `plans/` folders — Additional sections or content groups  

## Key Features

- **Responsive design** with layout adaption for different screen sizes  
- **Navigation menu** linking multiple pages  
- **Structured content sections** like about, activities, and plans  
- **External assets folders** maintaining organized project architecture  
- **Vanilla HTML/CSS/JS** (no external frameworks required)  
- **Ready for deployment** on GitHub Pages or any static host  

## Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript**
